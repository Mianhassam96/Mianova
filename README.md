
# 🎙️ Mianova

**Turn Text into Natural Voice Instantly**

Mianova is a modern and lightweight Text-to-Speech (TTS) web application that converts written text into natural-sounding speech. Built with React and the Web Speech API, it provides a clean interface, smart controls, and a smooth user experience designed for speed, accessibility, and simplicity.
 
## 🚀 Live Demo

[Mianova ](https://mianhassam96.github.io/Mianova)

## ✨ Features

Mianova allows users to instantly convert text into speech with simple play, pause, and stop controls. It supports multiple browser-based voices along with adjustable speed (rate) and pitch for a customizable listening experience. The app includes a Smart Play feature that automatically selects the best voice and settings, along with preset modes like Normal, Fast, Slow, and Podcast to simplify usage.

For better usability, Mianova includes a character counter, mini history for recently used texts, and the ability to save user preferences such as voice, rate, and pitch. A simple and advanced mode toggle ensures both beginners and advanced users have a smooth experience. The interface is clean, responsive, and enhanced with modern UI design and smooth transitions.

## 🛠 Tech Stack

Frontend: React.js
Styling: Tailwind CSS
Speech Engine: Web Speech API

## 📁 Project Structure

The project follows a scalable structure with separated components, hooks, and pages to keep the code clean and maintainable. Core speech functionality is handled inside custom hooks, while UI components are modular and reusable.

## ⚙️ Installation

Clone the repository, navigate into the project folder, install dependencies using npm install, and start the development server with npm run dev.

## 🧠 How It Works

Users simply enter or paste text into the input field and click Smart Play. Mianova processes the text and converts it into speech instantly. Users can further customize voice, speed, and pitch through advanced controls if needed.

## 🔮 Future Improvements
